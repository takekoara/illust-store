# Illust Store

Laravel + React (Inertia) を用いた  
個人開発のイラスト販売プラットフォームです。

## デモ
https://illust-store-production.up.railway.app/

## 概要
- 管理者が商品を投稿し、ユーザーが購入できるECサイト
- 認証・決済・リアルタイム通信を含むフルスタック構成
- 設計・リファクタリングの試行錯誤を含むポートフォリオ

## 技術スタック
- Laravel / PHP
- React / TypeScript
- Inertia.js
- Stripe
- PostgreSQL / SQLite

## 特徴
- 管理者権限による商品管理
- Stripe決済（テストモード）
- リアルタイム商品問い合わせ（Reverb）
- Service層によるビジネスロジック分離
- PHPUnitによるテスト（約200件）
