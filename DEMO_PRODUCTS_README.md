# デモ商品データ生成ガイド

推薦アルゴリズムのテスト用に、大量のダミー商品データを生成する機能です。

## 使用方法

### 基本的な使用方法

```bash
php artisan demo:generate-products
```

デフォルトで100個の商品が生成されます。

### 商品数を指定する場合

```bash
php artisan demo:generate-products --count=500
```

## 生成されるデータ

### 商品データ
- タイトル（日本語）
- 説明文（日本語）
- 価格（100円〜10,000円）
- 閲覧数（0〜5,000）
- 販売数（0〜200）

### タグ
以下のタグが自動的に作成・使用されます：
- オリジナル、女の子、男の子、背景、風景、キャラクター
- アイコン、デザイン、テクスチャ、装飾、和風、洋風
- ファンタジー、SF、動物、植物、建築、人物
- イラスト、素材、クリップアート、パターン、ベクター

各商品には1〜5個のランダムなタグが付与されます。

### エンゲージメントデータ
以下のデータも自動生成されます：

1. **いいね**: 各商品に0〜50個のランダムないいね
2. **ブックマーク**: 各商品に0〜30個のランダムなブックマーク
3. **閲覧履歴**: 各商品に0〜200件の閲覧履歴（過去30日間）

### 画像
プレースホルダー画像が設定されます。後で実際の画像に置き換えることができます。

## 注意事項

- 既存のデータは削除されません（追加のみ）
- 管理者ユーザーが商品の作者として使用されます
- 管理者ユーザーが存在しない場合は自動的に作成されます

## 推薦アルゴリズムのテスト

このデータを使用して、以下の推薦ロジックをテストできます：

1. **タグ類似度**: 共通タグが多い商品を推薦
2. **エンゲージメント指標**: いいね数、ブックマーク数、閲覧数が高い商品を推薦
3. **閲覧履歴ベース**: 過去に見た商品と類似した商品を推薦
4. **購入履歴ベース**: 購入した商品と類似した商品を推薦

## データの削除

生成したデータを削除する場合：

```bash
# 商品を削除（関連データも自動削除）
php artisan tinker
>>> App\Models\Product::where('title', 'LIKE', '%美しい風景イラスト%')->delete();
```

または、データベースをリセット：

```bash
php artisan migrate:fresh --seed
```
